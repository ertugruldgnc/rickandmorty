### How to Run the Project

To view the project in your browser using the **Live Server** extension, follow these steps:

1.  In the VS Code **Explorer** (left sidebar), click on the `rickandmorty.html` file to open it.
2.  Once the file is open, you have two ways to start the server:
    * **Option A:** Right-click anywhere inside the `rickandmorty.html` editor and select **"Open with Live Server"**.
    * **Option B:** Click the **"Go Live"** button located in the status bar at the very bottom-right corner of the VS Code window.
3.  Your default web browser will automatically open a new tab at `http://127.0.0.1:5500/rickandmorty.html`.
